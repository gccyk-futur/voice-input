# 产品需求文档（PRD）

## 语音助手（VoiceMate）

---

## 一、产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | VoiceMate（暂定） |
| 产品类型 | macOS 原生语音输入工具 |
| 核心价值 | 一键语音输入 + 可选 AI 润色，让口语秒变书面语 |
| 目标用户 | 文字工作者、学生、程序员、日常需要大量文字输入的用户 |
| 商业模式 | 完全免费，工具本身零成本，用户自带 API Key 或本地模型 |

---

## 二、产品定位

**“轻量、干净、可插拔”**

- 默认使用 macOS 系统听写（自动适配最新 API），开箱即用，零外部依赖
- 高级用户可自由选择 ASR 引擎（本地 Whisper / 云端 API）和 LLM 润色引擎（本地 Ollama / 云端 API）
- 不做云服务、不收集数据、不捆绑账号
- 所有密钥存储在本地 Keychain，隐私优先

---

| 角色 | 场景 | 期望 |
|------|------|------|
| 普通用户 A | 写文档、回邮件 | 按快捷键说话，文字直接填入，不想配置任何东西 |

## 四、功能需求

### 4.1 核心流程

```
用户按全局快捷键（默认 Cmd+Shift+V）
        ↓
弹出悬浮窗（显示实时识别文字）
        ↓
用户说话
        ↓
语音 → 文字（走选定的 ASR 引擎）
        ↓
文字实时显示在悬浮窗上半区（灰色）
        ↓
【如果启用润色】文字 → LLM 润色 → 逐字显示在下半区（黑色）
        ↓
用户按 Cmd+Enter 确认粘贴
        ↓
文字填入当前光标位置（模拟 Cmd+V）
        ↓
悬浮窗自动关闭
```

---

### 4.2 功能模块清单

#### 模块一：全局快捷键

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 默认快捷键 | Cmd + Shift + V | P0 |
| 自定义快捷键 | 用户可在设置中修改 | P1 |
| 冲突检测 | 检测是否与其他软件快捷键冲突并提示 | P2 |
| 实现方式 | 使用 Carbon `RegisterEventHotKey`（无需辅助功能权限） | P0 |

#### 模块二：悬浮窗

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 置顶显示 | 始终处于所有窗口最前方 | P0 |
| 半透明毛玻璃 | 不遮挡后方内容，视觉柔和（支持深色/浅色模式） | P0 |
| 实时识别区 | 上半区显示 ASR 原始识别结果（灰色） | P0 |
| 润色结果区 | 下半区显示 LLM 润色后结果（黑色，逐字出现） | P0 |
| 操作按钮 | 底部提供“取消”和“粘贴（Cmd+Enter）”按钮 | P0 |
| 拖拽移动 | 用户可以拖拽悬浮窗到任意位置 | P1 |
| 状态提示 | 显示“正在录音...”、“正在润色...”等状态 | P0 |
| 音量指示 | 显示麦克风实时音量波形（小圆点或条状） | P2 |

#### 模块三：ASR 引擎（语音转文字）

| 引擎 | 描述 | 依赖 | 默认状态 | 优先级 |
|------|------|------|----------|--------|
| **系统听写（自适应）** | macOS 原生语音识别<br>- macOS 26+ 使用 `SpeechAnalyzer`（更快更准）<br>- macOS 12–25 使用 `SFSpeechRecognizer` | 系统语言包（按需下载） | **默认启用** | P0 |
| **Whisper（本地）** | 基于 whisper.cpp（Apple Silicon 支持 Metal 加速） | 用户自行下载 GGML 模型文件（如 `ggml-base.bin`） | 可选（需手动配置路径） | P1 |
| 讯飞 | 讯飞开放平台语音识别 | 用户自填 AppID + APIKey | 可选 | P1 |
| 阿里云 | 阿里云智能语音交互 | 用户自填 AccessKey | 可选 | P2 |
| OpenAI Whisper API | OpenAI 的云端语音识别接口 | 用户自填 API Key | 可选 | P2 |
| faster-whisper | Python 实现（仅 Intel Mac 推荐） | 用户自行安装环境 | 可选（社区贡献） | P3 |

**引擎切换逻辑**：用户只能同时选择一个 ASR 引擎，切换后立即生效，无需重启 App。

> **关于本地模型下载**：Whisper 模型文件由用户自行决定是否下载，App 仅提供路径配置和提示，不强制捆绑。这既保持 App 体积轻量（<10MB），也尊重用户对磁盘和隐私的掌控。

#### 模块四：LLM 润色引擎

| 引擎 | 描述 | 依赖 | 默认状态 | 优先级 |
|------|------|------|----------|--------|
| **无（不润色）** | 直接输出 ASR 识别结果 | 无 | **默认** | P0 |
| **Ollama（本地）** | 本地 Ollama 服务，支持任意模型 | 用户安装 Ollama 并拉取模型（如 `qwen2.5:7b`） | 可选（默认选中） | P0 |
| OpenAI | GPT-4o-mini / GPT-4o 系列 | 用户自填 API Key | 可选 | P1 |
| DeepSeek | DeepSeek API（性价比高）<br>⚠️ 注意：`deepseek-chat` / `deepseek-reasoner` 将于 2026.7.24 停用，新模型为 `deepseek-v4-flash` | 用户自填 API Key | 可选（默认模型名需更新） | P1 |
| Claude | Anthropic Claude API | 用户自填 API Key | 可选 | P2 |
| 自定义 | 任意兼容 OpenAI 格式的 API Endpoint | 用户自填 URL + Key | 可选 | P2 |

**润色开关**：独立开关，关闭时 ASR 结果直接输出，不调用任何 LLM。

## 五、非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 系统听写模式下，从按键到悬浮窗出现 < 0.5 秒；悬浮窗动画帧率 ≥ 60fps；流式润色逐字延迟 < 200ms |
| 体积 | 基础版 App < 10 MB（不含 Whisper 模型） |
| 兼容性 | macOS 12 Monterey 及以上，Intel + Apple Silicon 双架构（Apple Silicon 优先使用 Metal 加速） |
| 隐私 | 所有 API Key 存储在本地 Keychain；不收集任何用户数据；不上报任何使用统计；所有数据仅在用户设备上流转 |
| 稳定性 | 录音崩溃不影响系统；热键注册失败时有友好提示；引擎切换平滑 |
| 国际化 | 首发简体中文，后续可扩展英文界面（系统语言自适应） |

## 六、用户操作路径

**路径一：开箱即用（零配置）**

```
下载 App → 拖进 Applications → 打开 → 授予麦克风权限 → 按 Cmd+Shift+V → 说话 → 文字填入
```
（系统听写语言包若未下载，系统会自动提示并引导下载）

**路径二：开启润色（Ollama 本地）**

```
下载 App → 安装 Ollama → 拉取模型（如 ollama pull qwen2.5:7b）→ 打开 App 设置 → 开启润色 → 确认 Ollama 地址 → 开始使用
```

**路径三：开启润色（云端 API）**

```
下载 App → 打开 App 设置 → 开启润色 → 选择 OpenAI/DeepSeek → 填入 API Key → 开始使用
```

**路径四：切换 ASR 引擎（本地 Whisper）**

```
下载 App → 自行下载 Whisper 模型文件（如 ggml-base.bin）→ 打开 App 设置 → ASR 区域 → 选择 Whisper → 指定模型路径 → 开始使用
```

**路径五：切换 ASR 引擎（云端 API）**

```
打开 App 设置 → ASR 区域 → 选择讯飞/阿里/OpenAI Whisper → 填入对应凭证 → 开始使用
```

## 七、异常处理

| 异常场景 | 用户提示 |
|----------|----------|
| 麦克风权限未授予 | 弹窗引导用户去“系统设置 > 隐私与安全性 > 麦克风”开启 |
| 辅助功能权限未授予（模拟粘贴需要） | 弹窗引导用户去“系统设置 > 隐私与安全性 > 辅助功能”添加 App |
| 系统听写不可用（语言包未下载） | 提示“系统听写不可用，请去系统设置 > 键盘 > 听写 下载中文语言包”（或引导至系统设置） |
| Ollama 服务未启动 | 提示“Ollama 服务未启动，请运行 `ollama serve`” |
| API Key 无效 | 提示“API Key 验证失败，请检查后重试” |
| 网络超时 | 提示“请求超时，请检查网络连接” |
| Whisper 模型文件不存在 | 提示“未找到模型文件，请检查路径，或下载模型后重试” |
| 快捷键冲突 | 提示“当前快捷键已被其他应用占用，请自定义修改” |

## 十、设计原则

1. **默认最小化**：开箱即用，零配置，普通用户无感知学习成本
2. **渐进增强**：高级功能对需要的人可见，不需要的人不干扰
3. **隐私优先**：数据只在用户自己控制的路径中流转，不上云、不跟踪
4. **模块可插拔**：每个引擎都可以独立替换或扩展，协议驱动
5. **用户掌控模型**：本地模型下载由用户决定，App 不强制捆绑任何模型文件

---

> **本 PRD 已根据 macOS 26（Tahoe）最新 API 和当前（2026年7月）主流引擎服务状态更新，确保技术方案时效性和可落地性。**
