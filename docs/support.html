VoiceKit

macOS 语音输入助手 — 全局热键，说话即输入

常见问题

Q: 热键不生效怎么办？
A: 请确保已在 系统设置 → 隐私与安全性 → 辅助功能 中授权 VoiceKit。

Q: 听写不工作？
A: 检查 系统设置 → 隐私与安全性 → 麦克风 和 语音识别 权限是否已授权。

Q: 如何配置 AI 润色？
A: 在设置 → AI 润色 中添加模型，支持兼容 OpenAI API 格式的云端服务和 Ollama 本地模型。

Q: 我的数据安全吗？
A: VoiceKit 是纯客户端工具，无后台服务器。语音数据直接传输到你配置的服务，不经过第三方。详见隐私政策。
